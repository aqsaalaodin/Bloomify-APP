# bloomify_fyp

Bloomify is a personal development and productivity app built with Flutter, designed to help users stay focused on their growth journey. The app allows users to create and manage daily tasks, track their progress, and gain insights into their personal achievements over time. With features like a customizable task system, a daily calendar view, progress analytics, and integration with Firebase for secure storage and authentication (Google and Facebook login supported), Bloomify offers a seamless and motivating experience.
